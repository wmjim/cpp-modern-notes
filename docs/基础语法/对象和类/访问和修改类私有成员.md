# 访问和修改类的私有成员

**私有成员**是类内访问权限最严格的成员，**只能在类的内部进行访问和修改**，外部无法直接访问。

类的私有成员仅对**该类的成员**及其**友元**可访问，无论这些成员位于相同还是不同的实例中。

## 获取器函数

获取器函数（也称为访问器）用于获取类的私有成员变量的值。

```cpp
class MyClass {
	double getLength() { return m_length; }
	double getWidth() { return m_width; }
	double getHeight() { return m_height; }
private:
	double m_length{ 1.0 };
	double m_width{ 1.0 };
	double m_height{ 1.0 };
};
```

- 按照约定，访问成员变量`m_member` 的成员函数通常命名为 `getMember()`
-  `bool` 类型的成员变量的获取器通常被命名为 `isMember()`

## 设置器函数

设置器函数（也称为修改器）用于设置类的私有成员变量的值。

```cpp
class MyClass {
	double setLength(double length) { if (length > 0) m_length = length; }
	double setWidth(double width) { if (width > 0) m_width = width; }
	double setHeight(double height) { if (height > 0) m_height = height; }
private:
	double m_length{ 1.0 };
	double m_width{ 1.0 };
	double m_height{ 1.0 };
};
```

- 更新该成员变量 `m_member` 的设置器函数通常命名为 `setMember()`
- 更新 `bool` 类型成员变量 `m_valid` 的设置器函数通常被命名为 `isValid()`
