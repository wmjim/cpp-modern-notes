# 函数

数（function）是把一些语句（函数体，function body）关联到一个名字和形参的 C++ 实体。

```cpp
#include <iostream>

// 函数原型/函数声明
int multiplyNumbers(int x, int y);

int main() {
	int num1 = 3, num2 = 7;
	int result = multiplyNumbers(num1, num2); // 函数调用
	std::cout << "The product is: " << result << std::endl;
	return 0;
}

// 函数定义
int multiplyNumbers(int x, int y) {
	int product = x * y;
	return product;
}
```

- 函数作为行为封装
- 参数传递控制副作用
- 生命周期保证安全
- `const` 写清契约
- 返回值解决所有权
- 函数也是值

## `main()` 的实参

```cpp
int main(int argc, char* argv[]) {
	// code
}
```

- `argc`：是从命令行输入字符串实参的个数，类型是 `int`。
- `argv`：是一个指针数组，指向命令行实参，包括程序名（`argv[0]`）。
  - `argv` 的最后一个元素（`argv[argc]`）总是 `nullptr`
  - `argv` 中的元素个数总是 `argc + 1`

```bash
./main -o 3
# argc = 3
# argv 包含 4 个元素 {"./main", "-o", "3", nullptr}
```
