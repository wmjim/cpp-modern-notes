# 结构体

`struct` 用于将多个不同类型的数据组织在一起。

```cpp
// 1. 声明结构体类型
struct Student {
	string name;
	int age;
	double score;	
};

// 2. 创建结构体变量
Student s1; 
s1.name = "Tom";	// 通过成员运算符 . 访问各个成员

// 3. 初始化结构体变量 c++11 支持列表初始化
Student s2 = {"John", 20, 88.0};
Student s3 {"Lili", 21, 90.5};
Student s4 {}; // 大括号内为空，结构体成员被设置为 0

// 4. 赋值结构体变量给另一个结构体变量
Student s5 = s2; 

// 5. 创建结构体数组
Student s[20];	// 创建一个包含 20 个 Student 类型的数组
Student s1[2] = {{"John", 20, 88.0}, {"Lili", 21, 90.5}}; // 结构体数组初始化
```
