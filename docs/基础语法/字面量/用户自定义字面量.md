# 用户自定义字面量

#纯右值 

**用户自定义字面量（User-Defined Literals, UDL）** 是允许程序员为整数、浮点、字符、字符串等字面量定义后缀，使其生成任意自定义类型或执行特定操作的机制。

| 类型  | 写法               | 示例         |
| --- | ---------------- | ---------- |
| 整数  | [[整型字面量]] ud-后缀  | `100_km`   |
| 浮点  | [[浮点字面量]] ud-后缀  | `1.23_Pa`  |
| 字符  | [[字符字面量]] ud-后缀  | `c`_X      |
| 字符串 | [[字符串字面量]] ud-后缀 | `hello`_id |

UDL 语法优先解析正常字面量；如果冲突则仍视为标准字面量（例如 `123LL` 不会触发自定义字面量）【支持数字分隔符 `'` #cpp14  】
