# 字符字面量

#纯右值 

字符字面量是在代码中使用 单引号 `' '` 包含的字符或字符序列，用于表达单个字符或多字符常量。

| 字符字面量                    | 类型                                  | 解释                | 示例      |
| ------------------------ | ----------------------------------- | ----------------- | ------- |
| `'c-char'`               | `char`                              | 普通字符字面量           | `a`     |
| `u8'c-char'`             | `char` #cpp20 前<br>`char8_t` #cpp20 | UTF-8 字面量 #cpp17  | `u8'a'` |
| `u'c-char'`              | `char16_t`                          | UTF-16 字面量 #cpp11 | `u'¢'`  |
| `U'c-char'`              | `char32_t`                          | UTF-32 字面量 #cpp11 | `U'猫'`  |
| `L'c-char'`              | `wchar_t`                           | 宽字符字面量            | `L'β'`  |
| `'multi-char sequence'`  | `int`                               | 多字符字面量            | `'AB'`  |
| `L'multi-char sequence'` | `wchar_t`                           | 宽多字符字面量 #cpp23    |         |

- `c-char`：可以是 [基本字符集](https://cppreference.cn/w/cpp/language/charset#Basic_source_character_set) 或 转义序列。
- `multi-char sequence`：两个或多个 `c-char`
